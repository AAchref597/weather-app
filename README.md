# 🌦 Weather App

A simple Python application that retrieves real-time weather information using the OpenWeather API.

---

## Features

- Search weather by city
- Current temperature
- Humidity
- Weather description
- Error handling for invalid cities

---

## Technologies

- Python 3
- Requests
- OpenWeather API

---

## Installation

Install the required package:

```bash
pip install -r requirements.txt
```

---

## Running the project

```bash
python main.py
```

---
